# Gait 2 Go: Gait Analysis Anytime, Anywhere 
## Gait 2 Go is a markerless gait analysis using MediaPipe Pose to increase the accessibility of gait analysis by only requiring a phone camera for use. Once a video (in the sagittal plane) of an individual walking is recorded, it is uploaded to the Gait 2 Go platform, where a joint angle report over the gait cycle is generated. 

### Gait 2 Go is a project developed by a team of four senior Biomedical Engineering students as part of their Senior Capstone Design.
