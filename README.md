# Gait 2 Go: Gait Analysis Anytime, Anywhere 
## Gait 2 Go is a markerless gait analysis platform that leverages MediaPipe Pose to make gait assessment more accessible using only a smartphone camera. Users record a sagittal-plane video of an individual walking, which is then uploaded to the Gait 2 Go platform to automatically generate a joint angle report over the gait cycle.

### Gait 2 Go is a project developed by a team of four senior Biomedical Engineering students as part of their Senior Capstone Design.
